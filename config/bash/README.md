# Install
Add the following lines somewhere in your `~/.bashrc`:
```shell
export TIMZWIEBEL_LINUX="<path_to_this_repository>"  # e.g., "${HOME}/linux"

export PATH="${PATH}:${TIMZWIEBEL_LINUX}/bin"

source "${TIMZWIEBEL_LINUX}/config/bash/.bashrc"
```
